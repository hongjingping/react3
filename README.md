# react3
React项目构建和路由
###### 第01节：构建：create-react-app 快速脚手架
1. `sudo cnpm install -g create-react-app` node版本在6
2. `create-react-app react_study`创建目录
3. 文件目录学习
  1. package.json
    1. 所需要的配置(详细package-lock.json)
    2. 命令
  2. public
    1. 公共文件(图标以及配置移动端应用等)
  3. src
    1. index.js是入口文件
    2. App.js页面文件
  4. node_modules 
    1. npm包
4. create-react-app太集成化，直接将我们的webpack也打包进了node_modules，无法修改
